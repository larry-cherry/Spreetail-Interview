# Spreetail Interview

Before proceeding make sure you have Node Version 14 or greater installed
### How to run project
1) Run with tsc-node
    - Install dependancies 
        - `yarn`
    - Start Project with yarn
        - `yarn start`
2) Build and run
    - Build index.js file
        - `yarn build`
    - Execute with node
        - `node index.js`
